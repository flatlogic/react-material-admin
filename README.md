## ðŸš€ Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/flatlogic/react-material-admin.git
   cd react-material-admin
   ```
2. **Install dependencies**
   ```bash
   yarn install
   ```
3. **Run the app**
   ```bash
   yarn start
   ```
4. **Runs the app in the development mode without backend**
    ```bash 
    yarn dev
    ```

5. **Build for production**
   ```bash
   yarn build
   ```
